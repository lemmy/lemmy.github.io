Eclipse Google Summer of Code BoF

Wayne plans to schedule BoFs based on popularity, so here's a little ad for the Eclipse Google Summer of Code BoF submission. The abstract currently reads like this:

In this BoF, we will discuss the Eclipse community's participation in the
Google Summer of Code (GSoC) 2008. We will discuss lessons learned from
past experiences, our processes for the coming session, attracting students 
and mentors, and more.

But what value might this BoF offer for you? Besides organizing our internal work, we are always happy to discuss your project proposals, welcome potential students and mentors or demo achievements from previous years. And because there are students involved, chances are high to have a beer or two. ;-) So make sure to leave a comment on the submission.

Btw. if this BoF doesn't get accepted, you can still see last year's results in Philippe's short talk "Summer of Code 2007 @ Eclipse: discover 21 cool NEW plugins projects in 10 minutes".
